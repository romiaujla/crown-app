# Crown App (WIP)

Crown is a multi-tenant management platform designed to support different business systems such as transportation management, dealership management, and inventory management.

## Tech Stack

- Frontend: React, TypeScript
- Backend: Node.js, TypeScript
- Architecture: Modular multi-tenant design

## Key Features (in progress)

- Multi-tenant system design
- Role-based access and configuration
- Scalable UI components and workflows
- Guided tenant onboarding flow

## Focus Areas

This project focuses on:

- Scalable architecture for multi-tenant systems
- Frontend performance and component design
- API-driven workflows and system integration

## Status

Work in progress and actively being developed.
